Project 3:

Project            : CCSC-Pricing-GPMA

Duration : April 2011- August 2013

Role  : Developer

Team Size : 4

Client  : 3M

 

 

 

Summary:

 

           3M is a well-known MNC Manufacturing large number of products. There are a very wide range of products manufactured by them. As a result the Pricing logic has to be different for each division as per each Business Logic. The Application GPMA takes care of integrating all the business logics for pricing  products and these prices  are send to other order management systems ,so that a product can be bought at a global common rate across different order systems.

 

        The Application acts as a hub getting the product, customer and contract information from different sources and using these data to prize the product based on specific requirements and logics. The Inbound data to the application happens through SSIS Jobs and after pricing the products the prices are send to other systems using Web Methods.

 

              As a Programmer Analyst, I have been trained in various modules of the GPMA application like the Product/Customer Addition and Removal module, setting up the necessary characteristics for a product module which is needed for arriving at a price for a product, requesting for a customized prize module, the email request to admin module and Approval module. I work closely with Senior Developers and clients in order to understand the change requests, impact analysis implementation and unit testing.
